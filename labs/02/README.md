2. Describe the Dataset

Title & Source - National Basketball Association Game Data, Opendatabay  

File Format - CSV File  
Size - 10.95 MB  
Basic statistics - Spielwerte eines NBA-Spiles von ob Zuhause oder Auswärts gespielt wurde, ob es ein Sieg oder Niederlage war und bis zu wie viele Punkte gemacht wurden.  
Geographic or temporal coverage - The data covers 28 seasons of NBA basketball, from the 1996-97 season through to the 2023-24 season. The earliest game date is 1 November 1996, and the latest is 17 June 2024. Data up to the 2022-23 season was collected in January 2024, with the 2023-24 season data collected in June 2024. The dataset includes information on various NBA teams and games played across different game types (regular season, playoff, play-in).
